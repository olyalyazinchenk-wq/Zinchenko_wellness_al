## 4. ФАЗА 2: AI-АВТОМАТИЗАЦИЯ (1-2 недели)

### 4.1 Подключить DeepSeek-разбор

**Схема работы:**

```
Клиент: заполнил анкету, загрузил анализы
  ↓
WellnessBot: пакует данные в контекст (анкета + анализы)
  ↓
DeepSeek API: промпт (base_review + специфический по направлению)
  ↓
AI: возвращает JSON {питание, дефициты, образ жизни, план}
  ↓
Сохранение в БД: AIReview (статус PENDING)
  ↓
Ольга в админке: проверяет AI-черновик, правит, подтверждает
  ↓
Бот клиенту: «Ваш разбор готов!» + красивый вывод
```

### 4.2 Что нужно сделать

| # | Задача | Использовать из nutrition_bot |

| 2 | 7 промптов в `prompts/` | `ai_prompts/*.txt` |
| 3 | Экшн «Запустить AI-разбор» в админке | — |
| 4 | Human review workflow: черновик → проверка → публикация | — |
| 5 | OCR для анализов через EasyOCR | `services/analysis_service.py` |

### 4.3 Конфигурация AI

```env
# .env
DEEPSEEK_API_KEY=sk-...           # ваш ключ
DEEPSEEK_MODEL=deepseek-chat      # модель

```
