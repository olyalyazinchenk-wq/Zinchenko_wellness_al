# Premium Russian Copy Editor

## Когда использовать
При проверке любых русскоязычных текстов проекта: ответы бота, меню, кнопки, лендинг, примеры досье, промпты, описания продуктов.

## Входные данные
- Русскоязычный текст для проверки.
- Контекст: для кого (клиент, оператор, модель), цель текста.

## Пошаговый workflow

1. **Англицизмы и кальки:** найти и заменить.
   - «имплементировать» → «внедрить»
   - «валидировать» → «проверить»
   - «конфигурация» → «настройка» (где уместно)
   - «дефолтный» → «стандартный»
   - «триггерить» → «вызывать / запускать»

2. **Шаблонность:** найти AI-штампы.
   - «Я бы рекомендовал...» → конкретное действие
   - «Важно отметить, что...» → убрать
   - «В современном мире...» → убрать
   - «Комплексный подход» → что именно
   - «Индивидуальный план» → в чём конкретно

3. **Ошибки:** грамматика, пунктуация, согласование, падежи.

4. **Тон:** премиальный, тёплый, конкретный, без пафоса и канцелярита.

5. **Соответствие линейке:** цены, названия продуктов, позиционирование.

## Критерии качества
- Убраны все англицизмы (кроме технических терминов).
- Убраны AI-штампы.
- Грамматических ошибок — 0.
- Тон соответствует премиальному продукту.

## Что запрещено
- Добавлять новые смыслы без согласования.
- Менять медицинские/юридические формулировки без safety-аудита.
- Превращать текст в рекламный спам.

## Формат результата
Исходный фрагмент → Исправленный фрагмент + комментарий (что исправлено).
