# Safety First Workflow

## Правило: safety перед скоростью

При ЛЮБОМ действии, затрагивающем клиентский опыт:
1. Проверить на диагнозы/лечение.
2. Проверить на красные флаги.
3. Проверить human review gate.
4. Проверить обработку плохих данных.
5. Проверить PD/privacy.

## Чек-лист
- [ ] Нет формулировок диагнозов («у вас X»).
- [ ] Нет назначений лечения («принимайте Y»).
- [ ] Нет обещаний эффекта добавок.
- [ ] Красные флаги → врач, не добавки.
- [ ] Human review перед клиентской выдачей.
- [ ] Плохой файл → запрос переслать.
- [ ] Нет персональных данных в отчётах/GitHub.
- [ ] Русский язык в клиентском интерфейсе.

## Red flags — немедленная эскалация
- Кровь, потеря сознания, острая боль.
- Суицидальные мысли.
- Дети/подростки без явного согласия родителя.
- Беременные/кормящие без особой осторожности.
