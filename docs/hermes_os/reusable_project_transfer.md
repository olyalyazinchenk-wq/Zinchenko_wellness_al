# Reusable Project Transfer

## Универсальные компоненты (можно переносить)
- `project-context-loader.md` — быстрый вход в любой проект.
- `task-packet-executor.md` — workflow «задача → отчёт».
- `knowledge-sync-reporter.md` — фиксация результатов.
- `agent-self-improvement-loop.md` — цикл улучшения навыков.
- `full-day-operating-cycle.md` — 8-фазный рабочий день.
- `token_economy_rules.md` — экономия ресурсов.
- `implementation_gate.md` — переход к реализации.
- `self_audit_protocol.md` — самопроверка.

## Проектно-специфичные (адаптировать)
- `wellness-safety-auditor.md` — медицинско-юридические правила.
- `lab-data-safety-reviewer.md` — работа с анализами.
- `premium-russian-copy-editor.md` — языковые нормы.
- `growth-and-offer-critic.md` — коммерческая специфика.

## Что НИКОГДА не переносить
- Секреты, ключи, `.env`.
- Клиентские данные.
- Медицинские правила без адаптации под домен.
- Проектные цены и продуктовую линейку.
