# Copy Skill Quickstart

Skill file:
- `ops/skills/pain-resonance-copywriter-skill.md`

## What It Does
Turns content into "this is my pain -> I feel understood -> I know next step" format.

## Manual Prompt Shortcut

Use this in chat when you need one strong post fast:

`Сгенерируй пост для @olga_nutri86 по навыку pain-resonance-copywriter-skill: тема <тема>, цель <цель>, CTA-уровень <soft/mid/strong>.`

## Daily Use
- Educational day: soft CTA
- Case day: mid CTA
- Conversion day: strong CTA ("хочу разбор")

## Quality Rule
If the first two lines do not trigger recognition ("это про меня"), rewrite hook first.
