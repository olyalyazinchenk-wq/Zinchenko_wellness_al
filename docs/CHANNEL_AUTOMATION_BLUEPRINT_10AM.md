# Channel Automation Blueprint (10:00 MSK)

Channel: `@olga_nutri86` ("О теле с душой")
Primary objective: convert trust into qualified dialogue and Premium Wellness Dossier demand.

## Why This Blueprint
Daily posting should be:
- stable (every day at 10:00)
- relevant (from approved content plan)
- conversion-aware (funnel stage logic)
- safe (no diagnosis claims, no fear pressure)

## Recommended Operating Model

### Stage 1: Strategy + Calendar
- weekly content map with funnel stages:
  - attention
  - trust
  - case
  - offer bridge
  - conversion

### Stage 2: Daily Production
- each day draft package contains:
  - post text
  - hook A/B
  - CTA line
  - first comment
  - optional visual brief (image or circle video)

### Stage 3: Publish Logic
Two modes:

1. **Approval Mode (recommended start)**
- 09:00: draft generated
- you approve quickly in chat
- 10:00: publish-ready final

2. **Autopublish Mode**
- 10:00: post published automatically to channel
- requires Telegram bot with posting rights in channel

### Stage 4: Feedback Loop
- daily metric checkpoint:
  - comments
  - saves
  - DM starts
  - "хочу разбор" trigger mentions
- if 2 weak posts in a row: rebuild hooks/CTA style

## Automation Package

Required automations:
1. `daily-post-generator-10am` (core)
2. `video-circle-planner` (Tue/Thu/Sat)
3. `channel-performance-audit` (daily evening)
4. `weekly-content-refresh` (Sunday)

## Publishing Asset Logic

For each day choose one:
- text only
- text + image brief
- text + circle video script

Rule:
- early growth phase -> prioritize text clarity + human circle videos

## Safety & Brand Guardrails
- no diagnosis claims
- no manipulative urgency
- clear escalation to offline care for red flags
- calm, warm, evidence-oriented voice

## Approval Gate (Must Confirm Before Final Setup)
- confirm target KPI
- confirm approval mode vs autopublish mode
- confirm bot/channel posting rights
- confirm allowed topics and taboo topics
