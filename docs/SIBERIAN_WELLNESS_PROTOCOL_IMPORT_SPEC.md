# Siberian Wellness protocol import spec

Цель: когда Ольга присылает PDF со схемами Siberian Wellness, превращать его не в свободный текст, а в безопасный справочник для бота.

## Как внедрять схемы

1. Извлечь из PDF названия программ и продуктов.
2. Разделить схемы по задачам: дефицитные риски, ЖКТ, желчеотток, микробиота, мягкая детокс-поддержка, антиоксидантная поддержка, сон/стресс, кожа/волосы, энергия.
3. Для каждой схемы указать условия применения: какие жалобы, какие анализы, какие ограничения.
4. Для каждой схемы указать стоп-факторы: беременность, ГВ, дети, онкология, активные аутоиммунные процессы, заболевания печени/почек/ЖКТ, камни в желчном, острые боли, температура, желтушность, серьёзные лекарства.
5. Не переносить из PDF медицинские обещания, если они есть. Формулировать как нутрициологическую поддержку.
6. В боте использовать формулировки: можно рассмотреть, может быть уместно, стоит обсудить, подходит как профилактическая поддержка.
7. Не писать: лечит, назначаю, схема лечения, гарантирует, убирает болезнь.
8. Если схема связана с желчеоттоком, сначала проверять красные флаги и необходимость УЗИ/врача.
9. Если схема связана с паразитарной гипотезой, не утверждать паразитов и не заменять диагностику.
10. В рекомендации добавлять реферальную ссылку/номер консультанта только после выбора конкретного продукта.

## Формат записи одной схемы

1. Название схемы.
2. Категория.
3. Для кого подходит.
4. Когда не использовать.
5. Какие данные нужны до рекомендации.
6. Продукты Siberian Wellness.
7. Альтернатива Vitamax, если есть.
8. Как объяснять клиенту простым языком.
9. Какие вопросы задать врачу.
10. Контрольная точка через 2-4 недели.

## Принцип для бота

Бот не назначает лечение. Бот подбирает нутрициологические ориентиры и программы поддержки по данным клиента: анкета, жалобы, питание, образ жизни, хронический фон, лекарства, добавки, анализы и противопоказания.
