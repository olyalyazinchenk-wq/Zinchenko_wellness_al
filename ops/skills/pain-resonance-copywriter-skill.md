# Pain Resonance Copywriter Skill

## Objective

Create channel content that makes the reader feel:
"This is my problem. I feel seen. I know what to do next."

The skill is built for trust-first wellness content with ethical sales bridge to Premium Wellness Dossier.

## When To Use

Use this skill for:
- daily Telegram posts
- video circle scripts
- conversion bridge posts
- comment starters under posts
- weekly recap posts

## Core Promise Formula

Every unit must include:
1. Problem recognition (mirror the reader state in plain language)
2. Emotional relief (reduce panic, increase clarity)
3. Practical next step (one clear action)
4. Bridge to offer (when relevant, never forced)

## Problem Mirror Framework

Before writing, detect 4 dimensions:
- Symptom layer: what body signal is visible
- Emotion layer: what person feels about it
- Social layer: what they hear from others ("у вас норма")
- Decision layer: what they do not understand now

Then write first two lines to mirror all four layers quickly.

## Post Construction (Text)

### Block A: Hook (1-2 lines)
- Must sound like the reader's inner phrase.
- No abstract intros.

### Block B: Clarifying Core
- Explain one mechanism in simple language.
- No long terminology chains.

### Block C: Practical Step
- Give one concrete, safe action.

### Block D: CTA Ladder
- Soft: comment one symptom
- Mid: request checklist in DM
- Strong: "хочу разбор" for premium stage days

## Video Circle Construction

Length: 30-60 sec

Structure:
1. "You are not crazy" moment (validation)
2. One key insight only
3. One next action
4. Warm CTA

## Sales Bridge Rules

Allowed:
- "Когда free clarity уже недостаточно..."
- "Если хотите, можно перейти к структурированному разбору."

Not allowed:
- pressure language
- fake urgency
- diagnosis promises

## Quality Checklist (Must Pass)

- Reader can identify own problem in first lines
- One idea per post
- Clear next action
- Calm and human tone
- No diagnosis claims
- CTA matches stage

## Anti-Patterns

Do not:
- write generic motivational text without practical value
- overload with markers and jargon in one post
- force premium CTA in every post
- use fear as conversion tool

## Auto-Refine Loop

If 2 cycles in a row show weak response (low comments, low saves, low DM starts):
1. Freeze current format.
2. Run mini-audit:
   - did hook mirror pain?
   - did action feel too vague?
   - was CTA too early or too hard?
3. Rebuild content pattern for next 3 posts.
4. Re-test and keep the better variant.
