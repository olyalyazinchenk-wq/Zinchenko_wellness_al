# Telegram Content Ops Skill

## Objective

Run a complete daily content operation for Telegram channel growth and sales funnel progression.

## Use For
- planning daily posts
- generating publish-ready drafts
- matching CTA to funnel stage
- preparing image/video briefs
- tracking post performance and improving next outputs

## Daily Workflow

1. Read approved content plan.
2. Identify today's funnel stage.
3. Generate post package:
   - hook A/B
   - body
   - CTA
   - first comment
4. Add optional asset:
   - image brief OR circle video script
5. Produce KPI target for the post.
6. After publish, log basic feedback and tune next day.

## Funnel Stage Rules
- Attention: relatability and pain recognition
- Trust: practical clarity and calm authority
- Case: concrete symptom pattern logic
- Offer bridge: explain when premium is relevant
- Conversion: strong but ethical CTA ("хочу разбор")

## Copy Constraints
- no diagnosis claims
- no fear pressure
- no generic motivation without practical step
- always include one clear action

## Performance Loop
If 2 low-response posts in a row:
1. Audit hook relevance
2. Audit CTA pressure level
3. Rebuild next 3 hooks with stronger pain mirror
4. Keep one winning variant and continue
